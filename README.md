Freelance Order Filter
Фильтрует заказы с фриланс-бирж по цене и ключевым словам, сохраняет результат в CSV и ведёт историю просмотренных заказов.

Что может делать:
- Ввод заказов в формате 'название;цена;сайт;описание'
- Фильтрация по минимальной цене
- Фильтрация по ключевым словам (поиск в названии и описании)
- Сортировка по цене (дорогие сверху)
- Сохранение результата в 'Orders.csv'
- История просмотренных заказов (повторы не показываются)

Как запустить
1. Установите Python 3.8 или выше
2. Скачайте файлы проекта
3. Запустите 'main.py'
4. Вставляйте заказы построчно, пустая строка - конец ввода
5. Введите минимальную цену и ключевые слова через запятую

Пример ввода:
Нужен бот для Telegram;5000;Kwork;Разработка бота для магазина
Парсинг сайта;1500;FL.ru;Собрать цены конкурентов
Дизайн;10000;Kwork;Разработка оформления для приложения

Требования
- Python 3.8+
